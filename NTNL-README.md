# NTNL reference

Aquest repositori és un fork de [`coreyhaines31/marketingskills`](https://github.com/coreyhaines31/marketingskills) mantingut com a **referència documental comuna per als agents NTNL**.

## Ús

- El fork conserva el repositori upstream complet.
- Les skills es consulten de manera selectiva segons la tasca.
- No és la skill operativa de cap agent i no s’han d’activar totes automàticament.
- Fermí manté `publicar_xarxes` com a skill operativa canònica; altres agents mantenen les seves adaptacions.
- Qualsevol adaptació local s’ha de fer a la skill de l’agent, no modificant les skills upstream sense una raó documentada.

## Skills prioritàries per al contingut NTNL

`product-marketing`, `customer-research`, `content-strategy`, `social`, `copywriting`, `copy-editing`, `marketing-psychology` i `lead-magnets`.

## Límits

- No incloure secrets, credencials ni informació privada de Colt o de la infraestructura NTNL.
- Verificar la llicència i el context de cada skill abans d’adaptar-la.
- Aquest document descriu l’ús del fork; no substitueix les instruccions específiques de cada agent.
